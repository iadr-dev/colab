## Adding New Libraries
Try: `/org/repo-name` (GitHub format).
Context7 indexes most public GitHub repos automatically.

## When multiple IDs exist, prefer in this order:
1. Official docs site IDs when they have better snippet coverage (often `/websites/...`)
2. Official source repo IDs with High reputation
3. Versioned IDs only when the task explicitly targets that version
