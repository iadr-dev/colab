# oh-my-colab — OpenCode Bootstrap

1. Run: npm install -g @iadr-dev/colab
2. Run: ohc setup
3. Read CLAUDE.md — this is your system context
4. Read .ohc/notepad.md at the start of every session

If npm unavailable: fetch https://raw.githubusercontent.com/iadr-dev/colab/main/templates/CLAUDE.template.md
and save as CLAUDE.md in the project root.
